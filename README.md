# Playwright TypeScript - warsztaty

Cześć!

To repozytorium szkoleniowe przygotowane na warsztaty **Automatyzacja testów funkcjonalnych aplikacji internetowych z użyciem Playwright i TypeScript**.

Podczas warsztatów będziemy pracować na przykładowej aplikacji TodoLab znajdującej się w katalogu `project` i ćwiczyć pisanie testów end-to-end w Playwright.

Warsztaty trwają **2 dni**.
