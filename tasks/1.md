## Zadanie nr 1

**Zapoznanie z projektem TodoLab**

1. zainstaluj zależności i uruchom aplikację:

```sh
npm install
npm run dev
```

2. otwórz `http://localhost:5173` i zaloguj się hasłem `admin123`
3. przetestuj ręcznie, zwróć uwagę na następujące funkcjonalności: dodawanie, edycję i usuwanie zadań; filtrowanie po statusie i priorytecie; wyszukiwanie po tytule/opisie; operacje masowe; import i eksport JSON; popup z raportem; routing `#/login`, `#/todos`, `#/about`

**Cel:** poznać aplikację przed pisaniem testów.
