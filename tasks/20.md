## Zadanie nr 20

**Composition pattern w Page Objectach**

Rozdziel odpowiedzialności:

- `LoginPage` - logowanie i błędy logowania
- `TodosPage` - lista, formularz, filtry
- `NavigationComponent` - menu i wylogowanie
- `TodoFormComponent` - dodawanie i edycja

Zastosuj composition pattern:

```ts
class App {
  readonly login: LoginPage;
  readonly todos: TodosPage;
  readonly navigation: NavigationComponent;
}
```

Przepisz test end-to-end na nową strukturę.
