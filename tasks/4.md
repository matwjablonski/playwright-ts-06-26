## Zadanie nr 4

**Konfiguracja środowiska testowego**

1. sprawdź konfigurację w `playwright.config.ts`
2. pobierz przeglądarkę, jeśli nie jest jeszcze zainstalowana:

```sh
npx playwright install chromium
```

3. uruchom test smoke: `npm test`
4. uruchom tryb UI: `npm run test:ui`
5. dodaj projekt mobilny `mobile-chrome` z `devices['Pixel 5']`
6. uporządkuj nazwy plików według konwencji: testy: `*.spec.ts`, Page Objecty: `*.page.ts`, setup: `*.setup.ts`, fabryki danych: `*.factory.ts`
7. dopisz w `README.md` krótką notatkę: jak uruchomić aplikację i testy.
