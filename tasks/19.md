## Zadanie nr 19

**Fixtures i większe zbiory danych**

Stwórz test fixtures dla różnych stanów TodoLab:

```ts
export const todoFixtures = test.extend<{
  todosWithMixedStates: void;
  overdueTodos: void;
  completedTodos: void;
}>({
  // przygotowanie localStorage przed testem
});
```

Napisz testy:

- lista z 50+ zadaniami
- wyszukiwanie w dużej liście
- operacje masowe na wielu zadaniach
- screenshot listy z różnymi priorytetami

Wskazówka: użyj `page.addInitScript()` do ustawienia `localStorage`.
