## Zadanie nr 22

**Typy dla testów Playwright**

Dodaj typy dla:

- Page Objectów
- fixtures
- helperów danych testowych
- odpowiedzi `/data.json`

Przykład:

```ts
type TodoFactoryOptions = Partial<Todo>;

function createTodo(overrides: TodoFactoryOptions = {}): Todo {
  // ...
}
```

Bonus: dodaj generyczny helper `mockJson<T>()`.
